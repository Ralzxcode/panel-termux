# Panel Termux

Panel CLI Termux berbasis Bash dengan login dan submenu.

## Cara Install
pkg update && pkg upgrade
pkg install git -y
git clone https://github.com/USERNAME/panel-termux.git
cd panel-termux
chmod +x panel.sh
./panel.sh

## Fitur
- Login password
- Menu & submenu
- Tampilan warna
- Aman (dummy)
